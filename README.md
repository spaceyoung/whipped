### 🔎 개요
반응형 웹을 구현할 수 있는 **CSS의 flex, 미디어 쿼리** 등의 기능 활용을 위해 **다양한 Viewport에 따른 화면 변화**에 중점을 두고 리디자인을 진행한 프로젝트입니다. 해당 프로젝트를 통해 **반응형 웹의 레이아웃 설계**를 익힐 수 있었습니다.
<br>
<br>
<br>
